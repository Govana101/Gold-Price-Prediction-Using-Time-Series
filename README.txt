Gold Price Prediction Using LSTM

This repository contains a project that predicts gold prices using Long Short-Term Memory (LSTM) neural networks. The model is trained on historical gold price data and aims to forecast future prices.

## Project Overview

The project involves the following steps:

1. Data Preprocessing
2. Data Visualization
3. Feature Scaling
4. Data Preparation for LSTM
5. Model Building and Training
6. Model Evaluation and Prediction
7. Visualization of Results

## Requirements

- numpy
- pandas
- matplotlib
- plotly
- scikit-learn
- tensorflow
- keras
